# 🌍 Travel Bot — Мадрид → Барселона → Рим → Милан

Персональный Telegram-бот для поездки 17–26 июня 2026.

## Структура файлов

```
travel_bot/
├── bot.py              # Точка входа
├── config.py           # Переменные окружения
├── db.py               # SQLite история диалогов
├── handlers.py         # Все команды и чат
├── prompts.py          # Системный промпт для GPT
├── knowledge_base.json # База данных поездки
├── requirements.txt
├── Procfile            # Для Railway
└── .env                # Твои токены (не публиковать!)
```

---

## Шаг 1 — Получи токены

### Telegram токен:
1. Открой @BotFather в Telegram
2. Напиши `/newbot`
3. Дай имя боту (например: `Spain Italy Trip`)
4. Получи токен вида `1234567890:AAF...`

### OpenAI ключ:
1. Зайди на platform.openai.com
2. API Keys → Create new secret key
3. Скопируй ключ вида `sk-proj-...`

---

## Шаг 2 — Локальный запуск

```bash
# Установи зависимости
pip install -r requirements.txt

# Создай файл .env
cp .env.example .env

# Открой .env и вставь свои токены:
# TELEGRAM_BOT_TOKEN=1234567890:AAF...
# OPENAI_API_KEY=sk-proj-...

# Запусти бота
python bot.py
```

---

## Шаг 3 — Деплой на Railway

1. Зайди на **railway.app** и зарегистрируйся
2. Нажми **New Project → Deploy from GitHub**
3. Загрузи папку `travel_bot` в GitHub репозиторий
4. В Railway: **Variables** → добавь:
   - `TELEGRAM_BOT_TOKEN` = твой токен
   - `OPENAI_API_KEY` = твой ключ
5. Railway автоматически запустит бота через `Procfile`

## Команды бота

| Команда | Описание |
|---------|----------|
| /start | Главное меню с маршрутом |
| /madrid | Мадрид: отель, места, транспорт, еда |
| /barcelona | Барселона: отель, места, транспорт, еда |
| /rome | Рим: отель, места, транспорт, еда |
| /milan | Милан: отель, места, транспорт, еда |
| /today | Программа на сегодняшний день |
| /budget | Полный бюджет поездки |
| /transport | Транспорт в текущем городе |
| /map | Карта текущего отеля |
| /help | Список команд |

Или просто пиши вопросы в чат — бот ответит как гид.
